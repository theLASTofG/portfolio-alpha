# Portfólio — João Vitor Gomes

Portfólio pessoal de programador com identidade visual inspirada em rock clássico e cultura musical vintage.

## Tecnologias

- HTML5 semântico
- CSS3 puro (Variables, Grid, Flexbox, Keyframes, Media Queries)
- JavaScript Vanilla (IntersectionObserver, Web Animations API)
- Google Fonts (Unbounded, Fraunces, Inter, IBM Plex Mono)

Sem frameworks ou bibliotecas externas.

## Como Executar Localmente

Abra o arquivo `index.html` diretamente no navegador ou use um servidor local:

```bash
# Com Python
python3 -m http.server 8080

# Com Node.js
npx serve .
```

Depois acesse: `http://localhost:8080`
